# v-component-dev

> vue组件开发脚手架

### 目录说明

```
|--demo             测试入口项目
    --main.js       测试项目入口，引入组件
    --App.vue       vue入口
|--src              组件入口
    |--components   组件模块
    |--assets       组件使用的资源文件
|--index.js         组件定义
```

### 安装
```javascript
npm install
```

### 开发
```javascript
npm run dev
```

### 编译
```javascript
npm run build
```

### 发布到npm
```javascript
npm add user
npm public
```
